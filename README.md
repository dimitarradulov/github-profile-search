## GitHub Profile Search
Gets github username from an input field and makes a call to the GitHub REST API and displays the profile name, image and bio. 

![GitHub Profile Search](./app-preview.PNG)

## Technologies used
- HTML
- CSS
- JavaScript
  - Fetch
  - GitHub REST API
